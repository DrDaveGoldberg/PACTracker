PoliticalModeling.com: PAC Tracker

This is a simple widget to let you explore the FEC filings of any committee, and see who they contributed to, and who their recipients contributed to and so on. At present, it's limited to the 2017-2018 campaign season. Simply start typing a PAC in the search form, and autocomplete. Then pull up the highlights on the link if you'd like more information or to switch to a view focusing on a different PAC.

If you'd like to increase the number of linkages, simply use a ?ngens=4 (for example), in the URL. If you'd like the change the number of individual PACs to be broken out separately, use ?ntop=10 (the default is 6).
